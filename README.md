# py-quiz
Take practice quizzes on Python and see your results.

## About
This is a simple Python script that allows you to take practice quizzes and see your results. 

## How to use

### Install dependencies
```
pipenv install
```

### Run the script
```
pipenv run python ./lib/pyquiz.py
```
